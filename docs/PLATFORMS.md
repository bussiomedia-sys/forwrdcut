# ForwrdCut — Platform Playbooks & Specs

Each placement is its own medium. Match the spec **and** the feel of that platform's best content.

## YouTube — in-stream skippable (TrueView) — PRIMARY ADS
- **Aspect:** 16:9 (1920×1080). **Sound:** ON. **fps:** 30.
- **The 5-second skip wall:** the Skip button appears at 0:05. Everything that matters to a
  skipper (scroll-stopper, reason to stay, brand on screen) lands before then.
- **VO is the spine**, big bold callouts reinforce (one idea per beat). Brand bug in by ~0:03.
- **Length:** :30 primary, :15 bumper cutdown (hook + strongest beat + end card). 15–30s is the
  sweet spot for selling the click; you can run longer if every second earns it.
- **End card:** black bg, wordmark/logo, URL, offer/warranty line, hold ≥2s.
- Build to Google **ABCD** (Attract / Brand / Connect / Direct).

## YouTube — Shorts
- **Aspect:** 9:16. **Sound:** ON. Native vertical, <1s hook, fast cuts, ≤60s. Same brand/CTA
  discipline; lighter touch than in-stream.

## Meta (Facebook/Instagram) — Feed & Reels ads — COLD TRAFFIC
- **Aspect:** 9:16 (1080×1920) + 4:5 (1080×1350). **Sound:** OFF by default → caption-first.
- **Hook:** scroll-stopping visual in <2s. Flow: hook → what it is → benefit/proof → offer+CTA.
- Risk reversal (e.g. 30-day money-back). Product-true claims only (NOT store-wide ratings on a
  single-product ad). `loudnorm` on. VO storytelling works well here too.
- Safe zones (9:16): keep captions off platform UI — `safe_top≈0.13`, `safe_bottom≈0.24`,
  `safe_side≈0.08`.

## TikTok / Instagram Reels — organic / creator
- **Aspect:** 9:16. **Sound:** ON. Authentic > salesy. Hook + payoff, lifestyle tone, keep the
  talent's own audio, soft brand sign-off. Music baked, but tell the human to swap a *trending*
  in-app sound. Gentle crossfades OK for travel/aesthetic; hard cuts for comedy. 7–34s.

## Landing-page hero loop (e.g. product-page top)
- **Aspect:** native (no reframe/crop). **Audio:** dropped. **No color grade** (true product
  color). No music. Premium feature callouts only — clean white chip + dark text + brand accent
  bar, fading in/out, timed to what's on screen. Desktop 16:9, mobile 1:1.

## Aspect-ratio handling cheatsheet
- Target 16:9 from **vertical** sources: cover-crop centered macros; blur-pad full-bag/full-body
  shots; reserve full-bleed for native-landscape clips. Shooting for ads? Capture **landscape
  hero/macro** coverage, not just vertical Reels footage.
- Target 9:16 from **landscape** sources: smart/subject-tracked reframe (face/subject detect) or
  center cover; avoid losing the subject.
